# 🤖 הוראות לקלוד — הקמת הסביבה במק

> **לגיא:** פתח טרמינל במק, הרץ `claude`, ואז כתוב:
> **"קרא את הקובץ הזה ובצע את כל מה שכתוב בו, אוטומטית."**
> קלוד יבצע את כל מה שלמטה. אתה רק תאשר פעולות ולפעמים תזין סיסמה.

---

## הקשר
גיא עבר מ-Windows למק. החבילה `מעבר-למק` (שירדה מ-OneDrive) מכילה:
- `דאשבורד/` — כל הפרויקטים (**בלי** node_modules ו-.venv — צריך לבנות מחדש)
- `claude-config/` — תוכן `~/.claude` מה-Windows (זיכרון, סקילים, סוכנים, הגדרות, חיבורי MCP)

⚠️ הפרויקטים **חייבים לרוץ מתיקייה מקומית, לא מתוך OneDrive** (סנכרון תוך כדי build = התנגשויות).
קודם כל מצא את נתיב החבילה (למשל `~/OneDrive/מעבר-למק` או `~/Downloads/מעבר-למק`) והשתמש בו בהמשך.

---

## משימה 1 — ההגדרות שלי (~/.claude)
1. אם כבר קיים `~/.claude` — גבה ל-`~/.claude.bak`.
2. `cp -R <נתיב-החבילה>/claude-config/ ~/.claude/`
3. ודא שקיים `~/.claude/projects/*/memory/MEMORY.md`.

## משימה 2 — הפרויקטים לתיקייה מקומית
```bash
mkdir -p ~/Projects && cp -R <נתיב-החבילה>/דאשבורד ~/Projects/דאשבורד
```
מכאן עובדים מ-`~/Projects/דאשבורד`.

## משימה 3 — כלי בסיס (התקן רק מה שחסר)
בדוק: `node -v`, `python3 --version`, `brew --version`.
- Homebrew: `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- `brew install node` (ל-campaign-center ולמחוללי הדאשבורד/PPTX)
- `brew install python` (לשני סוכני ה-Meta)
- `npm install -g @anthropic-ai/claude-code` (אם צריך)

## משימה 4 — בניית הפרויקטים

### campaign-center (Next.js — Campaign Center + AdPilot)
```bash
cd ~/Projects/דאשבורד/campaign-center
npm install
npm run dev          # http://localhost:3000
```
מחסנית: Next.js + Supabase + Gemini + Claude (claude-opus-4-8). טבלאות AdPilot בקידומת `ap_`. עברית RTL.
דורש `.env`/`.env.local` (Supabase + Gemini + Anthropic + Meta) — אם חסר, קח מה-`.env` בשורש `דאשבורד`.

### campaign-agent (Python / FastAPI — כלי AI פנימי)
```bash
cd ~/Projects/דאשבורד/campaign-agent
python3 -m venv .venv
source .venv/bin/activate          # ← במק זה bin/activate
pip install -r requirements.txt
```
הגדרות (ר' `README.md` בפרויקט): `.env` (`ANTHROPIC_API_KEY`, `META_ACCESS_TOKEN`) + `config/accounts.json`, `config/users.json`, `config/rules.json` (העתק מ-`*.example.json`). hash לסיסמה: `python -m backend.auth hash "סיסמה"`.

### meta-ads-monitoring-agent (Python — ניטור QA ל-Meta, קריאה בלבד)
```bash
cd ~/Projects/דאשבורד/meta-ads-monitoring-agent
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
משתמש ב-Supabase. דורש `ads_read` בלבד (לעולם לא `ads_management`). ר' `README.md`.

### שורש דאשבורד — מחוללי דאשבורד/PPTX ו-HTML
- `*_pptx.js`, `asif_dashboard.js` = סקריפטי Node (`node <file>.js`; אולי `npm install` בשורש).
- קבצי `.html` נפתחים ישירות בדפדפן.
- מחולל ההצעות: `proposal-editor.html` הוא המקור; אחרי עריכה הרץ `build-embed.js` כדי לחדש את `elementor-embed.html`.

## משימה 5 — תיקון נתיבי Windows
חפש `C:\Users\Guy\...` והמר לנתיבי מק (`~`/`/Users/...`) ב: `~/.claude/settings.json`, `~/.claude/projects/*/memory/*.md`, קבצי `.env`/קונפיג, וכל hook/סקריפט.

## משימה 6 — סקריפטי PowerShell
`GenerateMonthlyReport.ps1` לא ירוץ במק כברירת מחדל — התקן `brew install --cask powershell` (`pwsh`) או המר ל-bash/Python. שאל את גיא אם עדיין צריך.
